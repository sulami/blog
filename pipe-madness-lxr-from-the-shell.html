Pipe Madness: LXR from the Shell

Published: Do 05 Juni 2014

In Linux.

tags: pipemadness cli kernel lxr

Well, it is not really LXR, but it works on Linux-Next without actually having to use LXR and does something similar. The rather short pipe I use here is grepping a directory for a keyword and using awk and finally sort to present me with an ordered list of files which contain my keyword.

grep "keyword" -r ./*.c -c | grep -v 'Binary' | awk -F: '{if ($2!="0") print $2" "$1}' | sort -nr
