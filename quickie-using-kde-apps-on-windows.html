Well, yes, I occasionallyÂ  boot on Windows (8 Pro at the moment, but generally I prefer 7) to get some stuff done I cannot do on Linux, although I cannot really code on Windows, so I miss out there, too. One thing I like are cross-platform apps, such as Firefox, which I can use on both OSes. Those apps give me the possibilty to copy over my configs and have the same UX all the time. Most of the time, I prefer the programs coming from Linux, as those are generally better coded.

While I got KDE for Windows a while back, intending to use plasma instead of the Windows 8 shell, which has not worked out yet, I can use it to run standard KDE-apps such as konqueror, dolphin or amarok on Windows. They use the proper Qt-theme and work surprisingly well. The documentation is pretty bad, but installing should still be easy enough.
