Title: Projects

Here is some of my stuff:

* [FrankenWM]({filename}/pages/frankenwm.md)
  The small, dynamic tiling window manager I write and maintain
* [Nozdormu](https://github.com/sulami/nozdormu)
  An unittest-like framework for easy, precise and fast Python3 benchmarks
* [Feed2Maildir](https://github.com/sulami/feed2maildir)
  A simple solution to get RSS/Atom-feeds into a maildir automatically
* [hGM](https://github.com/sulami/hGM)
  A Yesod-based web tool for tabletop GMs and other writers
* [hcached](https://github.com/sulami/hcached)
  A drop-in replacement for memcached written in Haskell
* [Danketsu](https://github.com/sulami/danketsu)
  A simple, fast, event-based, RESTful microservice communication microservice
* [GoFuse](https://github.com/sulami/gofuse)
  A microservice component to prevent timeouts from killing your whole
  application
* [Elena](https://github.com/sulami/elena)
  A dynamic status API on Flask
* [Medusa](https://github.com/sulami/medusa)
  A really old, minimal nagios-compatible monitoring system in Python
* [wm](https://github.com/sulami/wm)
  A Wayland window manager in development
* [hmenu](https://github.com/sulami/hmenu)
  A cli fuzzy finder and launcher in Haskell
* [DGame](https://github.com/sulami/dgame)
  A demo of what can be done with D and OpenGL
* [Streamo](https://github.com/sulami/streamo)
  A shell script that allows for simple desktop streaming to Twitch.tv
* [pelican-theme](https://github.com/sulami/pelican-theme)
  The pelican theme you see here, based on my own 'cybercopy' colourscheme

I also currently maintain:
