2013-05-20 19:44 | in Web | tags: firefox

I prefer Firefox over Chrome because the customization is a huge deal for me, not only by Addons but also in terms of the general UI. I am using a handcrafted layout like this (sorry for the dropdown, screenshotting it is quite difficult):

Next to the overly long addressbar I have got a set of textless bookmarks for my most used sites. Then I use the bookmark menu for more specific pages to remember. The addressbar is still usable while using only half a screen.

Now to addons. I use some addons, currently nine active ones and 16 in total. Those include Adblock Edge, Disable Ctrl-Q Shortcut, DuckDuckGo Plus (yes, DDG is my primary search engine, the bang-commands are incredibely useful), Tab Wheel Scroll and Themed Button fo Linux.
