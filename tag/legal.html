When it comes to programming your own stuff, like many people do, at some point you might want to thing about licensing. If your code is just sitting on GitHub, you probably want to use an open source license, so eventually people can contribute easily to your project. But there are so many licenses, Apache, BSD, MIT, several GPLs, how is anyone supposed to keep track of what they do, and which one should you choose? Easy answer: TL;DR Legal.
