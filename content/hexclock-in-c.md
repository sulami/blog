Title: Hexclock in C
Date: 2014-06-23 14:15
Category: Coding
Tags: c, hexclock, hackernews

Earlier this weekend, I found [Hexclock][1] on hackernews, a simple website
which interpretes the time as hexadecimal color value and colors the whole page
using this value. I had nothing to do and decided to clone it in C and ncurses
for local use, [/r/unixporn][2] screenshots and stuff like this. Anyway,
[here][3] is the source, if you ask me really nice, I can also compile i686 and
x86_64 binaries for you.
